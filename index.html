Indoor Positioning Platform

Introduction

Overview

Indoor Positioning Platform is a Wi-Fi / UWB based indoor localization system for smart environments. It collects RSSI and channel information from multiple access points and estimates positions on a live dashboard.

Data are uploaded via MQTT to a cloud database (AliCloud / AWS). A web dashboard visualizes trajectory, heatmap and accuracy metrics for experiments and demos.

Experiment Notes

Testbed: 4 Wi-Fi APs @ 2.4 GHz, sampling RSSI per 100 ms. Data are cleaned and synchronized before fingerprinting. We evaluate average error (m), 90-percentile (m) and latency (ms).
