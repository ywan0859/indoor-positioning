Introduction

Overview

Smart Hospital Platform implements a complete IoT system that can be deployed in hospital scenarios. On the device side, data can be uploaded via Wi‑Fi, LoRa, UWB, or BLE, with gateways and edge nodes connected through the MQTT protocol.

Data are stored in cloud databases (e.g., AliCloud/AWS) and visualized through the web/mobile dashboards. You can replace the dashboard on the right with your own Aliyun DataV share link or any external web app.

How to customize

  1. Replace assets/hero.jpg with your own banner image (same file name).
  2. Edit texts in index.html.
  3. Replace the iframe src with your Aliyun DataV (or other) dashboard share URL.
